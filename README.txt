===================
AppDeployerUploader
===================


Info
===================

This app allow to uplaod files to AppDeployer application


Intallation
===================

	pip install -e git+https://github.com/jacek-marchwicki/PythonAppDeployerUploader.git#egg=app_deployer_uploader

Usage
===================

	app-deployer-uploader --help

Example:

	app-deployer-uploader --base-url="http://localhost:8888" 1003 6d3e13588ef0f4288405707fef19026ae49cbb532cfd9a21 0.0.1 README.txt
